# RoR Plus React <a name="about-project"></a>

**RoR Plus React** is RoR applications which has React as front-end framework. This test project can be used with educational purposes in order to get a functional project integrating this two important frameworks. This project implementes RoR and React in the same project.

- **Node.js & React implementation for project front-end development**
- **Ruby on Rails is used as framework on the back-end**
- **Postgresql used as database for the back-end**
- **The project has been implemented as a single application**
- **SPA development**
